# Lab 03: Two LANs Connected by a Router

## Objective
Connect two separate LANs using a router and verify inter-network communication.

## Topology
![Topology](topology.png)

## IP Addressing

### LAN 1
- Network: 192.168.1.0/24
- Gateway: 192.168.1.1

### LAN 2
- Network: 192.168.2.0/24
- Gateway: 192.168.2.1

## Router Configuration
- G0/0: 192.168.1.1
- G0/1: 192.168.2.1

## Verification
- Successful ping between LANs

## Lessons Learned
- Routers enable communication between different networks
- Each router interface connects to a unique subnet
